What I use:

HTML

HTML is the standard language for creating web pages. It provides the structure of a webpage and allows you to define elements like headings, paragraphs, images and links.

CSS

CSS is used to style and layout web pages. It allows you to control the colors, fonts, spacing, and overall design of a website, making it visually appealing. This is what I used to give this website some color and styling!

JavaScript

JavaScript is a programming language that enables interactive and dynamic content on web pages. It is used for tasks like form validation, animations, and creating interactive user interfaces. It is the cause for most animations in this website!

PHP

I use PHP to create dynamic web pages and interact with databases. It allows me to build functional back-end systems for websites and applications.

SQL

SQL helps me manage and manipulate databases efficiently. I use it to retrieve, insert, and update data, ensuring seamless integration with web applications.

C#

I enjoy using C# (C Sharp) for building Windows applications and eventually exploring game development. It allows me to create robust and interactive software solutions.

Git

I use Git for version control and collaboration on coding projects. It helps me track changes, manage code versions, and work effectively with other developers. It also allows me to share my projects and activities on GitHub.

Visual Studio

I use Visual Studio Code as my primary IDE for coding. It offers a powerful environment for writing, debugging, and testing code, making development more efficient.
